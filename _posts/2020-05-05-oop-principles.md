#### 캡슐화 : encapsulation

- 모듈화 : 독립적으로 역할을 제대로 수행하기 위해 필요한 데이터와 기능을 모은것
- 정보은닉 : 객체의 데이터 외부에서 접근하는 것을 막고 함수를 통해서만 조작할 수 있도록 한다 

#### 상속 : inheritance

- 재사용 : 상위 개념의 특징을 하위 개념이 물려 받는것

#### 추상화 : abstraction

- 모델링 : 객체들이 가진 공통의 특성을 파악하고 불필요한 특성을 제거한다

#### 다형성 : polymorphism

- 사용편의 : 모듈이 갖는 정체성과 표현방식, 추상 클래스나 인터페이스를 의미
